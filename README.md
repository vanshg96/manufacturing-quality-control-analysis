# Manufacturing Quality Control Analysis

This project analyzes manufacturing data to monitor product quality and detect potential defects using statistical methods.

## Features

- Data analysis of manufacturing process
- Statistical quality control techniques
- Visualization of production metrics
- Identification of defects and anomalies

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Project File

Manufacturing_quality_control.ipynb contains the full analysis and visualization of the manufacturing dataset.
